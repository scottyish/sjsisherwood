This is a non-exhaustive list of projects I've worked on

PhD thesis

This project focused on using model-based and multi-modal cognitive neuroscience techniques to increase our understanding of the role of the subcortex in motor and cognitive inhibition.

Stop Signal Task

This program runs the Stop Signal Task (SST), a task designed to test motor inhibition, auditory and visual versions are available. he experiment runs in Python using the exptools2 package, and the task events can be triggered by an MR pulse.

Multi-source Interference Task

This program runs the multi-source interference task (MSIT), a task designed to test cognitive bias and inhibition. The experiment runs in Python using the exptools2 package, and the task events can be triggered by an MR pulse.
